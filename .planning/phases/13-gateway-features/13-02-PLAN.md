  <action>
Add JWT auth generation to `internal/krakendgen/generator.go`:

1. **`buildAuthValidatorConfig(jwt *krakend.JWTConfig) map[string]any`** -- builds the auth/validator config map. Include non-zero/non-empty fields:
   - `"alg"`: jwt.GetAlg() -- always include if non-empty (default RS256 is common but must be explicit)
   - `"jwk_url"`: jwt.GetJwkUrl() -- include if non-empty
   - `"audience"`: jwt.GetAudience() -- include if len > 0 (this is a repeated string, serialize as JSON array)
   - `"issuer"`: jwt.GetIssuer() -- include if non-empty
   - `"cache"`: jwt.GetCache() -- include ONLY if true (false is default, omit it)
   - `"propagate_claims"`: call `buildPropagateClaims(jwt.GetPropagateClaims())` -- include if len > 0

2. **`buildPropagateClaims(claims []*krakend.ClaimToHeader) [][]string`** -- converts proto ClaimToHeader messages to KrakenD's expected array-of-arrays format: `[["claim_name", "Header-Name"], ...]`. Return nil if input is empty.

3. **`getJWTPropagatdHeaderNames(jwt *krakend.JWTConfig) []string`** -- extracts just the header names from propagate_claims. Used to auto-add them to input_headers.

4. **Update `buildEndpointExtraConfig`** to also check `gwConfig.GetJwt()`. If non-nil, add to extra_config map under `NamespaceAuthValidator`. JWT is service-level only -- read from gwConfig, never from epConfig.

5. **Update input_headers derivation in `GenerateService`** to auto-add JWT propagated claim headers. After calling `deriveInputHeaders(service, hm.method)`, check if `gwConfig.GetJwt()` has `propagate_claims`. If so, append each propagated header name to the input_headers slice (if not already present), then re-sort. This ensures KrakenD's zero-trust model is satisfied -- propagated claim headers must be in input_headers to reach the backend.

   Implementation pattern:
   ```go
   ep.InputHeaders = deriveInputHeaders(service, hm.method)

   // Auto-add JWT propagated claim headers to input_headers.
   if jwt := gwConfig.GetJwt(); jwt != nil {
       propagatedHeaders := getJWTPropagatedHeaderNames(jwt)
       if len(propagatedHeaders) > 0 {
           headers := ep.InputHeaders
           if headers == nil {
               headers = []string{}
           }
           for _, h := range propagatedHeaders {
               if !containsString(headers, h) {
                   headers = append(headers, h)
               }
           }
           sort.Strings(headers)
           ep.InputHeaders = headers
       }
   }
   ```

   Add a `containsString(slice []string, s string) bool` helper if not already present.

IMPORTANT: Use `NamespaceAuthValidator` constant, never inline `"auth/validator"`.
IMPORTANT: JWT is on `GatewayConfig` only. Do NOT read JWT from `EndpointConfig`.
